About PhytClust

PhytClust is a dynamic-programming-based algorithm for optimal phylogenetic tree clustering. It uses the CalBow scoring index with prominence-based peak detection to find the best number of clusters.

Please cite this repository if you use the algorithm in your work:
K. Ganesan, E. Billard, T.L. Kaufmann, C. B Strange, M. C. Cwikla, A. Altenhoff, C. Dessimoz, R.F. Schwarz, PhytClust (2025).
https://github.com/schwarzlab/phytclust

Acknowledgements: C. Robens, L. Godfrey, T. L. Kaufmann.

Help

Input format
  • Trees must be in Newick format.
  • Example: (A:0.1,B:0.2,(C:0.3,D:0.4):0.5);
  • Accepted extensions: .nwk .newick .new .txt .tree
Clustering modes
  • Auto — Finds optimal k using CalBow score peak detection.
  • Fixed k — Partitions the tree into exactly k clusters.
  • Multi-resolution — Finds one optimal k per log-spaced bin.
How to use
  1. Upload a Newick file or paste a tree string.
  2. Select clustering mode and set parameters.
  3. Click Run PhytClust (or press Ctrl+Enter).
  4. Explore results in the tabs.
  5. Use Export to download results.
Tree interaction
  • Click internal nodes to collapse/expand subtrees.
  • Right-click any node to rename, highlight, or resize.
  • Scroll to zoom, drag to pan.
  • Use the search box to find and highlight nodes by name.
Comparison mode
  • Enable "Compute all clusters" to use the Compare tab.
  • Select two different k values to see them side by side.
