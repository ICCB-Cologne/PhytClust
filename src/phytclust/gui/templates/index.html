Help

Input format

  • Trees must be in Newick format.
  • Example: (A:0.1,B:0.2,(C:0.3,D:0.4):0.5);
  • File upload accepts .new, .nwk, .newick or plain text files.

How to use

  1. Upload a Newick file or paste a Newick string.
  2. Set optional parameters (k, outgroup) as needed.
  3. Click Run PhytClust to compute clusters.
  4. View the Tree, JSON, and Logs in their tabs.
