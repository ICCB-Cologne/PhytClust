Help

1. Input format

  • Trees must be in Newick format.
  • Example: (A:0.1,B:0.2,(C:0.3,D:0.4):0.5);
  • File upload accepts .new, .nwk, .newick or plain text files.

2. How to use

  1. Upload a Newick file or paste a Newick string.
  2. Click Run PhytClust to compute clusters.
  3. View the JSON response and tree rendering.

3. Notes

  • Ensure your Newick string ends with a semicolon (;).
  • Very large trees may render slowly in the browser.
